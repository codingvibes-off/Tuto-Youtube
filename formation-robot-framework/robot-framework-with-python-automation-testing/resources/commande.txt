Command Line Scénarios

Excute All test in the folder : robot . 
Execute Specific test robot <testfilename>.robot
Execute Test Cases by Test Name robot --t <tagname> . (Tu dois mentionner le fichier ou se trouve le test)
Execute Test Cases By multiple tags robot --include <tagname1>AND<tagname1> . robot --include <tagname1>OR<tagname1> .
Execute the test by tag robot --exclude <tagname> .
Execute Tests by suite name robot --suite <foldername>
Execute Only Failed Tests robot --rerunfailed outpul.xml .
